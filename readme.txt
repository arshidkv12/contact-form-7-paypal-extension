Contact Form 7 - Integrate PayPal Submit button to Contact Form 7 which redirects directly to PayPal for payment. 

== Description ==

Most of the user face problems to integrate PayPal with Contact Form 7. ZealousWeb introduces a new plugin "Contact Form 7 - PayPal Extension" for those people. This Plugin will speedup Payment process by combining features of Contact Form 7 and PayPal at single step.

"Contact Form 7 - PayPal Extension" provides integration facility of PayPal fields with Contact Form 7. In current scenario, this concept is very useful for online payment. This plugin adds PayPal submit button that redirects to PayPal site with necessary information from Contact Form 7 and make the online payment after submission of your Contact Form 7. Our plugin helps to make faster online payment from PayPal with Contact Form 7. 

"Contact Form 7 - PayPal Extension" does not use the action handler 'on_set_ok' in the "Additional Settings" field or any other customization to Contact Form 7 plugin source code. User can easily add PayPal functionality to their Contact Form 7 by adding few parameters. This plugin requires Contact Form 7 to be activated.

= Contact Form 7 - PayPal Extension Features =

*	Multi-Currency: Supports 25 currencies with PayPal payment options
*	Multiple PayPal Account integration with every single form/button
*	Can get values from each fields of the form like textbox, drop-down, check-box etc.
*	Accepts dynamic value for item description and amount for each submission of contact form
*	Sandbox facility available for testing mode
*	Allows to add Return URL for successful and cancellation of payment


== Plugin Requirement ==

PHP version : 5.3 and latest
WordPress   : Wordpress 3.0 and latest

== Installation ==
1. Install "Contact Form 7 - PayPal Extension" either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin in your Wordpress Admin and go to edit "Contact Forms" at admin panel.
3. Choose "PayPal Submit button" from the Generate Tag dropdown.
4. Set all the parameters required to redirect to PayPal and make the payment.
